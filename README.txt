In order to run our source code to view the ToolMate website you must do the following:
- Ensure you have the latest version of Node.js and npm installed
- Navigate in a terminal to the root folder of our project
- Run 'npm install' to install the dependencies for our project
- Finally, run 'npm start' to run the project in a development web server that will open in the browser automatically.

Alternatively, you can visit "https://toolmate-43458.web.app/" to view the project.

It is not necessary to authenticate or login to use the project properly.
To properly test, try upvoting, downvoting, commenting, and replying to posts.
